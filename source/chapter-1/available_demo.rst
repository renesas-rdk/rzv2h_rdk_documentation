RZ/V ROS2 Demos
================

ROS2 packages designed for computer vision applications on Renesas RZ/V MPU platforms, specifically targeting the RZ/V2H.

The packages provide AI-accelerated vision capabilities including object detection, pose estimation, and visualization tools.

Available Demo Applications
--------------------------------

- :ref:`Vision Based Robotic Arm Teleoperation <arm_teleoperation>`: Vision-based control using MediaPipe hand tracking.
- :ref:`Vision Based Dexterous Hand <dexhand>`: Simultaneous mimic control of virtual and physical dexterous hands.
- :ref:`Rock Paper Scissors Game <rock_paper_scissors>`: Legacy RPS game using hand gesture recognition.
- :ref:`Static Object Detection <static_object_detection>`: General object detection using YOLOX models.
- :ref:`Static / Camera-based Hand Landmark Estimation <hand_landmark>`: Hand detection and landmark estimation using YOLOX and MediaPipe model.
